นำเข้าข้อมูลจาก Excel ลงฐานข้อมูล MySQL (Import from Excel (CSV) to MySQL Database) 

วิธีการติดตั้ง

1. ให้ ไปที่ Control Panel ของเว็บไซต์ของคุณ สร้างฐานข้อมูลชื่อ testdb (จะเป็นชื่ออื่นก็ได้นะ)
2. ใช้โปรแกรม Notepad (อยู่ที่ Start - All Programs - Accessories - Notepad) เปิด File ชื่อ dbconfig.php ครับ
3. ไปแก้ไข Code ภายในนี้ครับ (เป็นค่าที่เกี่ยวกับการเชื่อมต่อฐานข้อมูล ตั้งให้สอดคล้องกับที่สร้างไว้ในข้อ 1 นะ)
4. แก้เสร็จแล้วก็ Save ครับ (ไปที่ File -- Save นะ)
5. ทีนี้ให้คุณนำไฟล์ทั้งหมด UPLOAD ไปไว้บน Host ของคุณได้เลยครับ
6. ให้ใช้โปรแกรม FTP ทำการ CHMOD ไฟล์ .csv ที่จะนำเข้าเป็น 777 ด้วยครับ
7. ให้ใช้ BROWSE เปิดหน้าชื่อ setup.php เพื่อทำการสร้างตารางลงฐานข้อมูล (ใส่ URL ของไฟล์นี้ลงใน ADDRESS BAR ตัวอย่างเช่น http://www.sitename.com/exceltomysql/setup.php )
8. เป็นอันเสร็จสิ้นขั้นตอนการติดตั้งแล้วครับ สามารถทดลอง Import ข้อมูลได้โดยเปิดหน้า http://www.sitename.com/exceltomysql/import.php 
9. ใครมีข้อสงสัยอะไรก็อีเมล์มาถามกันได้นะครับที่ au_cty@hotmail.com ครับ
